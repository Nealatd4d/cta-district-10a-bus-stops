# CTA Bus Stops — School Board District 10A

Interactive map of all 465 CTA bus stops inside Chicago School Board District 10A.

## Features
- Interactive Leaflet map with clustered markers
- Search stops by name or street
- Filter by route (29 CTA routes) or direction
- District 10A boundary overlay
- Dark mode support
- Click any stop to see route details

## Data Sources
- [CTA Bus Stops](https://data.cityofchicago.org/Transportation/CTA-Bus-Stops/hvnx-qtky) via Chicago Data Portal
- District boundary constructed from [ward-precinct polygons](https://data.cityofchicago.org/Facilities-Geographic-Boundaries/Boundaries-Ward-Precincts-2023-/6piy-vbxa)
- Sub-district boundary verified via [CPS Board Member Finder](https://www.cps.edu/about/chicago-board-of-education/member-map/)

## Usage
Open `index.html` in a browser, or host on any static file server.
